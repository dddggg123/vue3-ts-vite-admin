# Vue3 TypeScript Vite Admin

### 项目介绍

1. 本项目用来实践和学习Vue3、Vite、qiankun等技术栈。
2. 帮助你快速搭建后台管理系统、免费且开源。
3. 本项目可拆分为1个主应用项目、2个微前端应用、1个iframe应用。
4. 项目结构简单便于二次开发和初学者学习。
